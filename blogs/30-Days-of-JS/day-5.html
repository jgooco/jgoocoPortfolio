August 10, 2022 • 3 min read

Finished Day 5! Let's recap!

So, what did I learn on Day 5 of 30DaysofJavaScript? Let's see!

Day 5 was all about arrays. I learend how to create them in JavaScript, how to accuess and modify them, the many methods to manipulate arrays, and arrays of arrays. I'll go through most of the things I learned, and things I found interesting! You can follow along or complete Day 5 yourself by clicking

Arrays

Unlike variables, arrays actually can store multiple values, each of which has an index in the array, and each index also references a memory address.

Arrays are ordered and can be modified. They can store duplicate elements as well as different data types.

How to Create an Array in JavaScript

Arrays can either be created empty or with values.

                    
const emptyArr = [] //creates empty array
const nonEmptyArr = [values, values, values] //creates an array with values

Accessing Array Items with Indexing & Element Modification

You can access an array element using indexing. An array index starts at 0, so if you want to access the first element, you would use arr[0]. To access the second element, you would use arr[1]. And so on. 

                    
const nonEmptyArr = [value1, value2, value3] //creates an array with values
nonEmptyArr[0]  //Accesses 1st element
nonEmptyArr[1]  //Accesses 2nd element
nonEmptyArr[2]  //Accesses 3rd element

You can use this access of element to modify the element itself like so:

                    
nonEmptyArr[1] = newValue  //Lets use the same array as above
console.log(nonEmptyArr) //would now print value1 newValue value3

Array Methods

This last section contained many methods one could use to manipulate arrays. I will go through a few of them, and if you want to learn about the rest, you can click here!

Fill

You can fill an array with elements of a static value using the fill method, as shown below:

                    
const arr = Array(10).fill('X')
//This here creates an array of 10 elements and fills each index with the string 'X'

Concat

Concantenates two arrays together.

                    
const firstArr = [1, 2, 3]
const secondArr = [4, 5, 6]
const concantArr = firstArr.concat(secondArr)
// This will create an array: [1, 2, 3, 4, 5, 6]

Length

Returns the length of the array.

                    
const arr = [1, 2, 3, 4, 5]
console.log(arr.length)

Includes

This method will check to see if an element is in the array. If it exiests, it will return true.

                    
const progLanguages = ['C++', C#, JavaScript, Java]
console.log(progLanguages.includes('C++')) // true
console.log(progLanguages.includes('Rust')) // false

Array Push and Pop

You can add and remove items at the end of an array using push and pop.

                    
const arr = ['element1', 'element2', 'element3']
arr.push('newElement') // makes array [element1, element2, element3, newElement]
arr.pop// removed newElement from array

Sorting Array

You can sort an array in ascending order using the sort method:

                    
const arr = [3, 2, 5, 1, 4]
arr.sort() // sorts to [1, 2, 3, 4, 5]
