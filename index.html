PROJECTS

Repurpose Gantry

Repurposed an engraver gantry via an Arduino Mega 2560 to control the stepper motors through joystick control for X, Y, and Z traversal. Rewired and soldered device to connect to Arduino board and external power supply. Attached a multimeter pin for testing purposes. Added a camera for as a replacement for a microscope.

C# GUI for Test Bench Form

Progammed a GUI in C# that could take the readings from a digital multimeter. Users open up a test bench sheet, fill out information, collect readings automatically or manually, and export the completed sheet.

CMPE 146 FreeRTOS MP3 Project

In a 3-person group, created an MP3 player using a C-Programmed FreeRTOS Microcontroller Board. Extracted MP3 files from onboard SD card through an external MP3 decoder via SPI communication and GPIO pins, created various FreeRTOS tasks for functionality, and displayed MP3 file meta data on external LCD screen via UART communication and GPIO pins.
